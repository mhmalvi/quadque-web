# Quadque Tech -v1.0

## Available Scripts

For working the development stage, in the project directory, you have to run:

### `npm run start-dev` or `yarn start-dev`

For working with production server, in the project directory, you have to run:

### `npm run start-prod` or `yarn start-prod`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

To Build with production server for the deployment use
### `npm run build-prod` or `yarn build-prod`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

For generating sitemaps there are an urls named [/build-sitemap](https://www.quadque.tech/build-sitemap). Click `Download Sitemap`. Inlcudes that in public folder and then build using `npm run build-prod` or `yarn build-prod`.
